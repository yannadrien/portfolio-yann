# Portfolio — Yann Adrien Maledy

## Démarrer en local
```bash
npm install
npm run dev
```
Ouvre l'URL affichée (souvent http://localhost:5173).

## Personnaliser
- `src/App.jsx` : contenu principal du site
- `public/MALEDY_YANN_ADRIEN_cv.pdf` : PDF du CV (déjà inclus)

## Build production
```bash
npm run build
npm run preview
```
